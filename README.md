# sao-workspace

> generator for express api with typescript.

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### Install workspace project

```bash
sao amirilovic/sao-workspace#main
```

### From git

```bash
sao amirilovic/sao-api#main packages/api
```
